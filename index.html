Projects

Work I've shipped and can walk through start to finish.

New York Search

A local search engine built in C++. Crawls 33 HTML files via recursive DFS, constructs an inverted index using std::map, and ranks results using dual-factor scoring: TF-IDF keyword density combined with backlink-based PageRank. Achieved zero rank error across all autograder test cases.

C++  ·  STL  ·  Recursion  ·  HTML Parsing  ·  Spring 2026

Inverse Word Search

A recursive backtracking solver that constructs letter grids satisfying up to 16 required words across a 10×5 search space with forbidden-word constraints. Replaced vector-based tracking with four 64-bit bitmasks, cutting solve time to under 8 seconds on the hardest test case.

C++  ·  Bitmasks  ·  Constraint Propagation  ·  Spring 2026

Document Analyzer

Parsed and analyzed text files to compute word sophistication, pair frequency, and language variety. Implemented Jaccard similarity on extracted word sets to compare writing style across documents.

Python  ·  NLP  ·  Sets  ·  Fall 2025
